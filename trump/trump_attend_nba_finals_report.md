- MSG: Madison Square Garden, 麦迪逊花园广场
- violent: 暴力的, 行凶的
    词根词缀：viol（力量 / 暴力）+ -ent（形容词后缀）
- affect: 影响；使改变
- regardless
    词根词缀：re-(加强) + gard (注视) + -less (无… 的)
    regardless of + 名词 / 代词 / 从句（最常用）
    不管；不顾；不论
    单独使用：无论如何；照样
- attendance: n 出席,到场
    词根词缀：attend (出席) + -ance (名词后缀，表状态 / 行为)
- courtside: 场边的
- celebrity:  名人；明星；名流
- with what to expect: 新闻常用表达：带来现场看点 / 前瞻报道（记者在场做赛前 / 现场解读）
- buzz: 热烈氛围 / 兴奋感 / 高涨人气（赛场、活动最常用）
- buzz kill: 【俚】煞风景的人 / 事、扫兴的事（你原句：a buzz kill 一件扫兴的事）
- understandable
- unfortunate: 不幸的, 令人遗憾的；可惜的
- According to: 根据；按照；据…… 所说
- coordination: 协作；配合；统筹安排
    词根词缀：co-(共同) + ordin (次序) + -ation (名词后缀)
    词根原形：coordinate
- Secret Service: United States Secret Service（USSS）美国特勤局
- presidential: 总统的；总统职位的
- special agent: 特工, 探员
- in charge: 核心含义：主管、负责、掌管
- special agent in charge: 现场负责人 主管特工
- in part: 短语：部分地；其中一部分, 指后面只是他发言内容的节选，并非完整讲话。
- ensure: 确保, 保证
    词根：en-(使) + sure (确定的)
- spokesperson: 发言人
- permit: v:允许；准许 n:许可证；特许文件
- pack: 挤满、塞满（强调人非常多，摩肩接踵）
- plaza: 广场、露天集散场地
- wrap:
    完整常用表达：wrap up（口语 / 体育场景高频）
    wrap (up) = 结束、收官
- wrapped with a win = 比赛落幕，球队取胜 / 赛事以胜利告终
- rowdy: 释义：喧闹的、吵吵闹闹的；（人）起哄闹腾的, 本句形容人群欢呼喧闹、场面热闹嘈杂。
- barricades: 
    词根：barrier（屏障）衍生词
    释义：路障；护栏；隔离围栏（安保、人群管控常用）
    用法 & 搭配
    metal barricades 金属护栏
    set up barricades 设置路障 / 隔离栏
- tight security
    tight 此处表示严密的、戒备森严的
    tight security 严密安保（新闻 / 赛事高频搭配）
- be held in: 在…… 举办 / 举行, 固定被动句式，表 “活动在某地举行”
- brooms: 扫帚
- security briefing:
    释义：安保简报；安全工作通报
    security 安保、安全
    briefing n. 简报、简短部署会、情况通报
    give a briefing 做简报
